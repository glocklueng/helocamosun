Airwülf is an autonomous helicopter project. The idea is to modify a standard hobby-style RC helicopter so that it flies itself. Airwülf will receive commands wirelessly from a base station PC/laptop. Once the command is received and interpreted, Airwülf will make all the calculations necessary to carry out the instructions. Airwülf's on-board computer wil make all the control surface changes according to its fuzzy logic algorithm.
